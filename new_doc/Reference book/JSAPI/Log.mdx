# log print debugging information

Log prints debugging information, which is convenient for locating problems.

## Example

log print location `logs/application.log` Print log:

```javascript
log.Error("%v", foo);
log.Info("foo");
```

The following information is output:

```
time="2022-08-25T10:00:28Z" level=error msg=foo
time="2022-08-25T10:00:28Z" level=info msg=foo

```
