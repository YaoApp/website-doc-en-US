# Save

保存单条记录。如数据记录中包含主键字段则更新，不包含主键字段则创建记录；返回创建或更新的记录 ID。

## `save` 保存单条记录, 不存在创建记录, 存在更新记录, 返回记录 ID

### 处理器

`models.模型名称.Save`

### 参数表

| 参数    | 类型       | 说明     | 示例                                               |
| ------- | ---------- | -------- | -------------------------------------------------- |
| args[0] | Object Row | 数据记录 | `{"name": "用户创建","manu_id": 2,"type": "user"}` |

### 返回值

`Integer` 创建或更新的记录 ID

## 示例一

### 数据模型

| 模型 | 模型定义                              |
| ---- | ------------------------------------- |
| user | [模型描述文件](../examples/user) |

### 处理器

```json
models.user.Save
```
