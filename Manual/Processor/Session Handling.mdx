## handler list

Application engine related processor.

| Processor     | Description                    | Documentation |
| ------------- | ------------------------------ | ------------- |
| session.start | start session return sessionID | -             |
| session.set   | Set session data               | -             |
| session.Get   | Get session data               | -             |

## code example

Set the value of the session, create a new `flows/session.flow.json`, the three parameters of args correspond respectively: session key, session value, expiration time

```json
{
  "label": "session",
  "version": "1.0.0",
  "description": "session",
  "nodes": [
    {
      "name": "SID",
      "process": "session.start"
    },
    {
      "name": "Set session data",
      "process": "session.set",
      "args": ["user", "123456", 28800]
    }
  ],
  "output": "success"
}
```

After setting the session, you can use `Process("session.Get", "user")` to get the corresponding value
